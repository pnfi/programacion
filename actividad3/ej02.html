Ejemplos Practicos de JavaScript

Esta Hoja contiene ejemplos de ejercicios básicos sobre lo que se puede hacer con javascript

Ejemplo 02 - mostrar una alerta
para mostrar una ventana de alerta se usa la palabra reservada alert 

Ejemplo:
<script type="text/javascript">
	alert("Oye esta es una alerta!!!")
</script>
eso va a generar una ventana de alerta como la de hace un momento.
Nota: las alertas no reciben codigo HTML.
