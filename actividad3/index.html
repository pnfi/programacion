Resumen
El lenguaje .js es muy parecido a php y muy necesario para hacer que las páginas webs tengan interacción con el usuario. para lograr esto se deben estudiar las diferentes palabras reservadas y tener en cuenta las posivilidades para validaciones que se pueden hacer.
---
Un dato curioso del lenguaje JavaScript es que tiene muchos hermanos o primimos asi como Ajax, y todos estos lenguajes son regulados por ECMAScript, quien es el encargado de que todos los lenguajes de script web tengan un estandar y puedan ser interpretados por los navegadores.
--- Entre otras cosas tambien pude investigar un poco y darme cuenta de que el lenguaje de javascript se ejecuta totalmente en el computador del usuario y por eso paginas como youtube y facebook pueden poner muy lentas a computadoras de baja gama o viejas las cuales si notan la diferencia al abrir estos sitios que contienen muchos script.
---
Entre otras cosas el nombre de JavaScript fue un prestamo en los inicios de este lenguaje, ya que mozilla se habia unido a Oracle SUN y esta empresa bien consolidada les dejo usar el nombre de java + Script para que los usuarios confiaran mas en el lenguaje y darle mayor peso de valides. Pero como ya sabemos este lenguaje no tiene nada que ver con java. y esa es la razon resumida.
