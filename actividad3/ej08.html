var, let, constant (Contenedores de variables)
Las variables son contenedores que pueden ser declarados con las palabras reservadas de:
var ==> Se usa como una variable local y externa, entre las variables y funciones.
let ==> Se usa como una variable local, solo funciona en cascada dentro de una variable o función.
const ==> Este contenedor no funciona como una variable, sino como una constante y no puede ser sustituido su contenido.
Ejemplo:
Para ver el ejemplo se debe ver el resultado desde la consola del navegador. Usar console.log
Código: 

<script type="text/javascript">
    var mensajeVar = ('Mensaje en consola con contenedor var ');
    let mensajeLet = ('Mensaje en consola con contenedor let ');
    const mensajeConst = ('Mensaje en consola con contenedor const');
    console.log(mensajeVar + mensajeLet + mensajeConst);
    console.log(mensajeVar);
    console.log(mensajeLet);
    console.log(mensajeConst);
</script>
